# Tentang Proyek

Proyek ini merupakan bagian dari **portofolio pribadi Gizelle Mathilda Mamentu** yang dikembangkan sebagai media untuk menunjukkan kemampuan dalam pengembangan perangkat lunak, mulai dari perancangan antarmuka, pengembangan frontend, hingga implementasi backend dan teknologi pendukung.

Selain sebagai dokumentasi hasil pengembangan, proyek ini juga menjadi sarana untuk mengeksplorasi teknologi modern serta menerapkan praktik pengembangan aplikasi yang terstruktur, responsif, dan mudah dipelihara.
